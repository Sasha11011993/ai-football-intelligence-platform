# Visual References

These images are the approved visual targets for the first implementation phases.

## Files

- `dashboard-target.png` - target look for the main analytics dashboard.
- `case-study-target.png` - target look for the portfolio Case Study page.

## Usage

Implementation phases should treat these images as design references, not as static assets to copy into the UI.

Phase 1 should use them to guide:

- layout density;
- navigation structure;
- Ukrainian-first labels;
- red accent usage;
- dark theme look;
- AI Analyst and Agent Trace positioning;
- Automation Hub and Data Flow presentation;
- Case Study page structure.

Light theme should be implemented as a polished alternate state even though these references show the dark theme.
